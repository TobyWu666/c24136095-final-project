# 量子計算期末專案報告

**課程：** 量子計算
**學號：** c24136095
**主題：** 含雜訊量子演算法之態斷層重建，以及量子核方法支援向量機（QSVM）

---

## 一、專案簡介

本專案以 [Qiskit](https://www.ibm.com/quantum/qiskit) 為主要工具，涵蓋三個量子計算的子題，分別探討「含雜訊環境下的量子演算法表現」與「量子機器學習」兩大主軸：

| 題目 | 主題 | 重點 |
| :--- | :--- | :--- |
| 問題一 | 雙位元 Grover 搜尋演算法 | 在含雜訊後端執行，並以量子態斷層掃描重建密度矩陣，計算保真度（Fidelity） |
| 問題二 | 三位元量子傅立葉轉換（QFT） | 對糾纏初態執行 QFT，重建密度矩陣並計算糾纏量（Negativity） |
| 問題三 | 量子核支援向量機（QSVM） | 以兩種特徵映射對 Iris 資料集分類，比較分類準確率 |

問題一、二著重於「真實量子硬體上雜訊如何破壞理想量子態」，問題三則展示「量子核方法」在機器學習分類任務上的應用與比較。

## 四、方法說明

### 問題一：含雜訊的雙位元 Grover 搜尋

- **目標：** 搜尋被標記的解 `f(3)=1`，理想輸出量子態為 `|11⟩`（即十進位的 3）。
- **電路組成：** 均勻疊加（Hadamard）→ Oracle（以 CZ 對 `|11⟩` 做相位翻轉）→ Diffuser（擴散運算子，放大目標振幅）。
- **雜訊模型：** 採用 IBM 假後端 `FakeManilaV2`，透過 `AerSimulator.from_backend` 載入真實裝置的雜訊特性。
- **態斷層掃描：** 對每個量子位元分別在 X、Y、Z 三個基底量測，每個量測設定執行 8192 次（shots），共 9 種設定（3²）。再以線性反演重建密度矩陣，並投影至最接近的合法物理態（Hermitian、半正定、跡為 1）。
- **評估指標：** 以保真度 `F(ρ, ρ′)` 衡量重建態與理想態 `|11⟩⟨11|` 的接近程度。

### 問題二：含雜訊的三位元量子傅立葉轉換（QFT）

- **初態：** 先製備 GHZ 糾纏態 `(|000⟩ + |111⟩)/√2`，再施加三位元 QFT。
- **態斷層掃描：** 三個位元各取 X、Y、Z 基底，共 27 種設定（3³），每設定 8192 shots。
- **評估指標：**
  - **保真度：** 重建態與理想 QFT 輸出之接近程度。
  - **糾纏量（Negativity）：** 以部分轉置（partial transpose）後負特徵值的絕對值總和，量化三種不同二分割下的糾纏強度。

### 問題三：量子核支援向量機（QSVM）

- **資料集：** 優先讀取 `data/` 內由 Moodle 提供的訓練/測試 CSV；若不存在，則退而採用 `sklearn.load_iris()` 的可重現分層 70/30 切分。
- **特徵編碼：** 將 4 個特徵縮放至 `[0, π]`，作為旋轉角度，使用 4 個量子位元編碼。
- **兩種特徵映射比較：**
  - **模型 A — ZZ 特徵映射：** Qiskit 內建的 `ZZFeatureMap`。
  - **模型 B — 硬體高效率特徵映射（HEA）：** 自行建構的 RY/RZ 資料編碼層加上 CX 糾纏層。
- **量子核：** 使用 `FidelityStatevectorKernel`（精確態向量，等同理想無雜訊）計算核矩陣。
- **分類器：** `sklearn.svm.SVC(kernel="precomputed")`，以測試集準確率為評估指標。

## 五、實驗結果

### 問題一：雙位元 Grover（含雜訊）

| 指標 | 數值 |
| :--- | :--- |
| 理想輸出態 | `\|11⟩ = \|3⟩` |
| 保真度 Fidelity `F(ρ, ρ′)` | **0.9259** |
| `\|11⟩` 態佔據機率 | 0.9259 |
| 糾纏量 Negativity | 0.0 |
| 轉譯後電路深度 | 17 |
| 量測設定數 / shots | 9 / 8192 |

> **說明：** 保真度約 0.93，顯示即使在含雜訊環境下，Grover 演算法仍能高機率地找到正確解 `|11⟩`。糾纏量為 0 符合預期，因為理想輸出 `|11⟩` 是可分離的乘積態（product state），本身不含糾纏。

### 問題二：三位元 QFT（含雜訊）

| 指標 | 數值 |
| :--- | :--- |
| 理想輸出態 | `QFT[(\|000⟩ + \|111⟩)/√2]` |
| 保真度 Fidelity | **0.737** |
| Negativity（A=q0 \| B=q1,q2） | 0.0769 |
| Negativity（A=q1 \| B=q0,q2） | 0.1491 |
| Negativity（A=q2 \| B=q0,q1） | 0.1725 |
| 轉譯後電路深度 | 30 |
| 量測設定數 / shots | 27 / 8192 |

> **說明：** 保真度約 0.74，明顯低於問題一。這是因為 QFT 電路較深（深度 30 vs. 17）、且使用較多雙位元閘，累積的雜訊更多。三組 Negativity 皆為正值，代表重建出的態仍保有可量測的糾纏結構，與 GHZ 糾纏初態的性質一致。

### 問題三：QSVM 分類（Iris 資料集）

- **資料來源：** `sklearn.load_iris()` 後備資料（未偵測到 Moodle 提供之 CSV）。
- **類別：** setosa / versicolor / virginica（三類）。
- **資料規模：** 訓練 105 筆、測試 45 筆，4 個特徵（量子位元數 = 4）。

| 模型 | 測試集準確率 |
| :--- | :--- |
| 模型 A：ZZ 特徵映射 | 0.8000 |
| 模型 B：HEA 特徵映射 | **0.9111** |

> **說明：** 在本資料集上，自建的 HEA 特徵映射（91.1%）優於內建的 ZZ 特徵映射（80.0%）。由混淆矩陣可知，HEA 模型對 setosa 類別達到 100% 正確分類，主要誤判集中於 versicolor 與 virginica 之間（此二類在特徵空間中本就較難區分）。

## 七、結論

1. **雜訊對深層電路影響顯著：** 比較問題一（深度 17，保真度 0.93）與問題二（深度 30，保真度 0.74），可清楚看出電路越深、雙位元閘越多，累積雜訊使保真度下降越明顯。
2. **態斷層掃描可量化雜訊：** 透過多基底量測重建密度矩陣，能將「肉眼看不見」的雜訊影響具體化為保真度與糾纏量等可比較的指標。
3. **特徵映射設計影響量子機器學習表現：** 在 Iris 分類任務中，自建的 HEA 特徵映射優於內建的 ZZ 特徵映射，顯示特徵編碼方式的選擇對 QSVM 的分類能力至關重要。
